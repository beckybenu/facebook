# Benu Studio — Site vitrine

Site vitrine animé pour l'entreprise de développement web & mobile, construit
avec **React + Vite + Tailwind CSS + Framer Motion**.

## Animations

- **Préloader** multilingue avec rideau de sortie animé
- **Hero** : révélation du titre lettre par lettre, dégradé animé, orbes en parallaxe au scroll
- **Curseur personnalisé** avec ressort (spring) et mix-blend-mode
- **Barre de progression** de scroll en haut de page
- **Marquee** infini des technologies
- **Cartes services** : apparition en cascade au scroll, icônes rotatives au survol
- **Cartes projets** : effet tilt 3D qui suit la souris
- **Compteurs animés** pour les statistiques
- **Timeline** de méthode avec ligne qui se dessine au scroll
- **Bouton magnétique** qui attire le curseur dans la section contact
- Grain filmique animé sur toute la page, respect de `prefers-reduced-motion`

## Lancer le projet

```bash
npm install
npm run dev        # développement (http://localhost:5173)
npm run build      # build de production dans dist/
npm run preview    # prévisualiser le build
```
