L'équipe recrutée

Bienvenue. Donnez une mission à NeuralStarK — il la confiera à l'agent compétent.
Exemples : « Prépare 3 posts LinkedIn pour la semaine », « Ajoute un rendez-vous vendredi 10h avec Dupont », « Rédige un email de relance pour le client Martin ».

🔒 Version 100 % navigateur : vos données restent dans ce navigateur (localStorage).
🆓 Mode automatique activé par défaut : aucune clé API nécessaire (un compte gratuit Puter est proposé à la première mission). Vous pouvez aussi brancher Claude, Gemini, Groq, OpenRouter… via la pastille en haut à droite.

🔑 Activer le cerveau

Choisissez un fournisseur d'IA. Votre clé est stockée uniquement dans votre navigateur (localStorage) et n'est transmise qu'au fournisseur choisi, directement depuis cette page. Sans clé, le tableau de bord reste utilisable en mode manuel.
